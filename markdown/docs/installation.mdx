# 系统安装指南

## 系统环境要求

在安装 PaYphp 码支付系统前，请确保您的服务器满足以下要求：

* **服务器配置**：推荐 2核2G 或更高配置
* **PHP 版本**：8.0 或更高版本
* **MySQL 版本**：5.7 ~ 8.0
* **Web 服务器**：Nginx 1.26 或 Apache 2.4

> 💡 **推荐**：使用宝塔面板进行环境配置，简单易用。
> 服务器推荐选择阿里云 ECS 或腾讯云 CVM 云服务器。

## 安装步骤

### 1. 环境准备

1. 安装宝塔面板
2. 在宝塔面板中安装 LNMP 环境（Linux + Nginx + MySQL + PHP）
3. 确保 PHP 版本为 8.0，并安装必要的 PHP 扩展：
   - fileinfo
   - curl
   - mysqli
   - pdo_mysql
   - openssl
   - mbstring

### 2. 下载安装包

1. 从官方渠道获取 PaYphp 码支付系统安装包
2. 解压安装包到本地

### 3. 上传文件

1. 使用 FTP 工具或宝塔面板上传功能
2. 将解压后的文件上传到网站根目录

### 4. 配置数据库

1. 在宝塔面板中创建新的 MySQL 数据库
2. 记录数据库名、用户名和密码
3. 导入系统提供的 SQL 文件

### 5. 修改配置文件

1. 找到 `config/database.php` 文件
2. 修改数据库连接信息：
   ```php
   'mysql' => [
       'host' => '数据库主机地址',
       'database' => '数据库名',
       'username' => '用户名',
       'password' => '密码',
       'charset' => 'utf8mb4',
       'collation' => 'utf8mb4_unicode_ci',
       'prefix' => '',
   ],
   ```

### 6. 设置目录权限

确保以下目录可写：
- `/runtime`
- `/public/uploads`
- `/config`

### 7. 配置 Web 服务器

#### Nginx 配置

```nginx
location / {
    try_files $uri $uri/ /index.php?$query_string;
}
```

#### Apache 配置

确保 `.htaccess` 文件存在于网站根目录。

### 8. 访问后台

1. 访问 `http://您的域名/admin`
2. 默认管理员账号：admin
3. 默认密码：123456
4. 首次登录后请立即修改密码

## 常见问题

如果您在安装过程中遇到问题，请参考以下解决方案：

1. **500 错误**：检查 PHP 版本和扩展是否正确安装
2. **数据库连接失败**：确认数据库配置信息是否正确
3. **权限问题**：检查目录权限设置

## 注意事项

* 安装完成后请立即修改默认管理员密码
* 定期备份数据库和重要文件
* 关注官方渠道获取系统更新信息

> ⚠️ **重要提示**：请仔细阅读文档，遇到问题先查阅文档解决。如需技术支持，请通过官方渠道联系。
