# 部署教程

## 🎉 创建站点

### 1. 添加新站点
- 进入控制面板
- 点击左侧菜单 "网站" → "添加站点"

### 2. 配置站点信息
- 输入已解析的域名
- 选择 PHP 版本：8.0
- 确认信息无误后点击提交

### 3. 验证站点
- 打开新的浏览器标签页
- 访问配置的域名
- 出现提示页面即表示创建成功

## ✨ 上传程序

### 1. 获取安装包
- 访问授权站下载专属安装包
- 将源码压缩包保存到本地

### 2. 上传文件
1. 进入宝塔面板
2. 选择左侧菜单 "网站"
3. 进入新建站点的根目录
4. 点击左上角 "上传" 按钮
5. 选择本地的 peak程序.ZIP 文件
6. 等待上传完成后关闭窗口

### 3. 解压文件
- 将压缩包解压到当前根目录
- 耐心等待解压完成
