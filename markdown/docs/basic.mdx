# PayPHP 码支付系统基础功能

本节将详细介绍 PayPHP 码支付系统的基础功能，帮助您了解系统的核心功能和日常操作。

## 控制面板

控制面板是系统的核心界面，提供了系统概览和快速访问入口。

### 控制面板概览

控制面板包含以下主要区域：

- **数据统计** - 显示交易金额、订单数量等关键指标
- **最近交易** - 展示最近的交易记录
- **系统状态** - 显示系统运行状态和告警信息
- **快捷操作** - 提供常用功能的快速访问入口

### 使用技巧

- 可以自定义控制面板的布局和显示内容
- 点击统计图表可以查看详细数据
- 使用筛选功能可以快速找到需要的信息

## 支付管理

支付管理是系统的核心功能，用于处理支付订单和交易记录。

### 创建支付订单

1. 进入"支付管理" → "创建订单"
2. 填写订单信息：
   - 商品名称
   - 订单金额
   - 支付方式
   - 订单备注
3. 点击"生成订单"按钮
4. 系统生成支付二维码或支付链接
5. 用户扫码或点击链接完成支付

### 订单管理

在"支付管理" → "订单列表"中，您可以：

- 查看所有订单及其状态
- 搜索和筛选订单
- 查看订单详情
- 关闭未支付的订单
- 对已支付的订单进行退款操作

### 退款处理

1. 在订单详情页面，点击"申请退款"按钮
2. 填写退款信息：
   - 退款金额
   - 退款原因
   - 退款备注
3. 点击"确认退款"按钮
4. 系统处理退款请求，并显示退款状态

## 商户管理

商户管理功能用于管理系统中的商户信息和账户。

### 商户信息管理

在"商户管理" → "商户列表"中，您可以：

- 查看所有商户及其状态
- 添加新商户
- 编辑商户信息
- 禁用或启用商户

### 商户账户管理

在"商户管理" → "账户管理"中，您可以：

- 查看商户账户余额
- 处理提现申请
- 调整账户余额
- 查看账户流水记录

## 订单管理

订单管理功能提供了对系统中所有订单的管理能力。

### 订单查询

在"订单管理" → "订单查询"中，您可以：

- 按多种条件搜索订单
- 导出订单数据
- 批量处理订单

### 订单统计

在"订单管理" → "订单统计"中，您可以：

- 查看不同时间段的订单统计
- 分析订单趋势
- 生成统计报表

## 通知管理

通知管理功能用于配置和管理系统的通知方式。

### 通知配置

在"通知管理" → "通知配置"中，您可以设置：

- 邮件通知
- 短信通知
- 微信通知
- 站内消息

### 通知记录

在"通知管理" → "通知记录"中，您可以：

- 查看所有通知记录
- 重新发送失败的通知
- 测试通知配置

## 日志管理

日志管理功能用于查看系统的操作日志和错误日志。

### 操作日志

在"日志管理" → "操作日志"中，您可以：

- 查看管理员的操作记录
- 按用户、操作类型等条件筛选日志
- 导出日志数据

### 系统日志

在"日志管理" → "系统日志"中，您可以：

- 查看系统运行日志
- 查看错误和警告信息
- 分析系统问题

## 使用技巧

为了更高效地使用系统的基础功能，我们建议：

1. **熟悉快捷键**：使用键盘快捷键可以提高操作效率
2. **使用批量操作**：对多个订单或商户进行批量处理
3. **定制显示列**：根据需要调整列表中显示的字段
4. **使用筛选器**：创建和保存常用的筛选条件
5. **设置自动通知**：配置重要事件的自动通知
