# 开始使用 PayPHP 码支付系统

欢迎使用 PayPHP 码支付系统！本节将帮助您快速了解系统并开始使用。

## 什么是 PayPHP 码支付系统？

PayPHP 码支付系统是一个专为个人和小型商户设计的轻量级支付解决方案，支持多种支付方式，包括微信支付、支付宝等。系统具有简单易用、安全可靠、功能丰富等特点。

## 系统要求

在开始使用 PayPHP 码支付系统之前，请确保您的服务器满足以下要求：

- PHP 7.4 或更高版本
- MySQL 5.7 或更高版本
- Nginx 或 Apache 服务器
- SSL 证书（用于安全支付）
- Composer 依赖管理工具
